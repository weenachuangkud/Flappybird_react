# Flappybird_react
Flappy bird clone website using react.

# How to run
1. Clone the repo
```bash
git clone https://github.com/weenachuangkud/Flappybird_react.git
```
2. Go into the project folder and install dependencies
```bash
cd Flappybird_react
npm install
```
3. Start the development server
```bash
npm run dev
```
